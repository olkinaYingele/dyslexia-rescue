# קורא — Reader App
### Приложение для чтения текста вслух для детей с дислексией

---

## Цель
Помочь ребёнку с дислексией (13 лет, не умеет читать) усваивать школьную программу.
Приложение фотографирует доску или учебник и читает текст на иврите вслух.

---

## Платформа
- **iOS** (приоритет) — iPhone 12 Pro Max и новее
- **Android** — следующий этап

---

## Функциональные требования

### 1. Захват изображения
- [ ] Съёмка камерой прямо из приложения
- [ ] Выбор фото из галереи
- [ ] Поддержка рукописного текста (доска)
- [ ] Поддержка печатного текста (учебник, рабочий лист)

### 2. Распознавание текста
- [ ] Распознавание иврита (рукописный + печатный)
- [ ] Разбивка текста на абзацы/блоки
- [ ] Сохранение порядка чтения (справа налево, сверху вниз)
- [ ] Обработка смешанного контента (текст + числа + формулы)

### 3. Выбор абзаца
- [ ] Список абзацев пронумерован
- [ ] Тап по абзацу → начинается чтение
- [ ] Активный абзац подсвечен

### 4. Чтение вслух
- [ ] Голос Carmit (иврит, he-IL)
- [ ] Подсветка текущего **слова** в процессе чтения
- [ ] Кнопки: Пауза / Продолжить / Стоп
- [ ] Звук работает при включённом беззвучном режиме

### 5. Экран чтения
- [ ] Крупный шрифт (≥32pt)
- [ ] RTL-выравнивание (справа налево)
- [ ] Жёлтая подсветка текущего слова
- [ ] Кнопка "назад к абзацам"

---

## Нефункциональные требования
- Интернет нужен только для распознавания текста (Claude API)
- Приложение работает на бесплатном Apple ID (для теста)
- Для школьного использования нужна подписка Apple Developer ($99/год)
